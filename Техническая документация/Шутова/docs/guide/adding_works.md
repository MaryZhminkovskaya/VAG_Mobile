# Добавление работ в выставку

В этом разделе описаны все способы добавления работ в ваши выставки.

## Добавление новой работы

**Процесс загрузки новой работы:**

1. Откройте нужную выставку
2. Нажмите кнопку "Добавить работу"
3. Выберите "Загрузить новую"
4. Заполните информацию о работе:
   - Название
   - Описание (по желанию)
   - Изображение
5. Нажмите "Сохранить"

Работа будет автоматически добавлена в выставку и появится в вашем профиле.

## Добавление существующей работы

Если работа уже есть в вашем профиле, вы можете добавить её в выставку:

1. Откройте выставку
2. Нажмите "Добавить работу"
3. Выберите "Добавить существующую публикацию"
4. Найдите нужную работу в списке
5. Отметьте её и подтвердите выбор

**Важно:** одна работа может находиться в нескольких выставках одновременно.
