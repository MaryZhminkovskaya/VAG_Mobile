# Добро пожаловать в Виртуальную Арт-Галерею

Добро пожаловать в документацию по приложению **"Виртуальная Арт-Галерея"** — вашему цифровому пространству для искусства.

Данная документация предлагает ознакомиться с таким объектом нашего сервиса, как **публикация**.

**Каждая публикация в системе включает:**

1. Визуальное представление
2. Детальное описание
3. Категории
4. Авторскую информацию
5. Историю просмотров 
6. Количество лайков

---

*Каждая публикация может принадлежать к:*

- [x] **Персональной коллекции** автора
- [x] **Множеству выставок** одновременно
- [x] **Множеству категорий**
- [x] **Понравившимся публикациям**
- [ ] **Виртуальным выставкам** (в разработке)
