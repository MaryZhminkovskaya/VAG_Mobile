# Создание публикаций

*Процесс добавления новых художественных работ в галерею*

## Предварительные требования

*Перед созданием публикации убедитесь, что работа соответствует:*

### Технические требования
- [x] **Формат файла**: JPEG, PNG, WebP
- [x] **Максимальный размер**: 10 МБ
- [x] **Минимальное разрешение**: 800px по меньшей стороне
- [ ] **Цветовой профиль**: sRGB рекомендуется

### Содержательные требования
- [x] **Авторская работа** — создана вами лично
- [x] **Качественное изображение** — четкое, без артефактов
- [x] **Соответствие тематике** — художественное произведение
- [ ] **Отсутствие водяных знаков** — они добавятся автоматически

## Процесс создания

*Для создания новой публикации:*

1. **Перейдите** в раздел "Мои публикации"
2. **Нажмите** кнопку "Создать публикацию"
3. **Заполните** [форму публикации](/forms/)
4. **Отправьте** на модерацию

*Среднее время модерации составляет 4-8 часов*
