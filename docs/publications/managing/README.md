## Удаление публикаций

*Процесс удаления работ из системы:*

### Этапы удаления
- [ ] **Подтверждение действия** — модальное окно с предупреждением
- [ ] **Проверка зависимостей** — участие в выставках, комментарии
- [ ] **Окончательное удаление** — без возможности восстановления

### Последствия удаления
- [x] **Работа исчезает** из всех выставок
- [x] **Удаляются метаданные** из поиска
- [x] **Освобождается место** в вашем хранилище
- [x] **Уведомляются подписчики** об удалении
