# Система публикаций

*Основные принципы работы с художественными работами в галерее*

## Концепция публикаций

**Публикация** в Виртуальной Арт-Галерее — это *цифровое представление* вашего художественного произведения, которое:

- [x] Сохраняется *перманентно* в вашем профиле
- [x] Может участвовать в *неограниченном количестве* выставок
- [x] Содержит *полные метаданные* о работе
- [x] Проходит *процесс модерации* перед публикацией

## Категории публикаций

*В системе поддерживаются следующие категории работ:*

### По формату
- [x] **Живопись** и графика
- [x] **Цифровое искусство**
- [x] **Фотография**
- [x] **Смешанная техника**

### По статусу
- [x] **Черновики** — неопубликованные работы
- [x] **На модерации** — ожидающие проверки
- [x] **Одобренные** — доступные в галерее
- [x] **Отклоненные** — видимые только автору, могут быть изменены или удалены

## Жизненный цикл публикации

*Типичный путь работы в системе:*

1. **Создание** → Заполнение [![Форма публикации](https://www.unite.ai/wp-content/uploads/2024/12/AI-in-Art.png)](/publications/creating/)
2. **Модерация** → Проверка модератором (4-8 часов)
3. **Публикация** → Появление в галерее
4. **Участие** → Добавление в выставки
