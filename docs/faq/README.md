### Что такое публикация и чем она отличается от выставки?

**Публикация** — это *отдельное произведение*, которое вы загружаете в галерею. Каждая публикация имеет:

- [x] Собственную страницу с описанием
- [x] Изображение работы в высоком качестве
- [x] Метаданные (название, техника, год создания)
- [x] Возможность участвовать в multiple выставках

**Выставка** — это *тематическая коллекция публикаций*, которая может содержать:

- [x] Работы одного автора или нескольких художников
- [x] Произведения, объединенные общей темой или концепцией
- [x] Публикации разных лет создания

*Проще говоря: публикация — это картина, а выставка — это галерея, где она висит.*

---

### Какие требования к загружаемым изображениям?

*Для успешной публикации работа должна соответствовать техническим требованиям:*

#### Форматы файлов
- [x] **JPEG** (.jpg, .jpeg) - оптимально для фотографий
- [x] **PNG** (.png) - рекомендуется для графики и живописи
- [x] **WebP** (.webp) - современный формат с хорошим сжатием
- [ ] **GIF** (.gif) - анимированные изображения не поддерживаются

#### Технические параметры
- [x] **Максимальный размер**: 10 МБ
- [x] **Минимальное разрешение**: 800px по меньшей стороне
- [x] **Рекомендуемое разрешение**: 1500-4000px для детализации
- [x] **Цветовой профиль**: sRGB

### Сколько времени занимает модерация работы?

*Стандартное время модерации зависит от нескольких факторов:*

#### Очередь модерации
- [x] **Обычное время**: 2-6 часов
- [x] **Пиковая нагрузка**: до 12 часов
- [x] **Ночное время**: может занять до 24 часов

#### Что проверяют модераторы:
- [x] **Техническое качество** изображения
- [x] **Соответствие тематике** платформы
- [x] **Авторские права** и оригинальность
- [x] **Корректность метаданных**

*Вы получите уведомление о результате модерации.*

### Что делать, если работа не прошла модерацию?

*Если работа отклонена, вы получите уведомление с причиной. Возможные сценарии:*

#### Технические проблемы
- [ ] **Низкое качество** изображения → перезагрузите в лучшем качестве
- [ ] **Неверный формат** → конвертируйте в поддерживаемый формат
- [ ] **Превышен размер** → уменьшите размер файла

#### Содержательные проблемы
- [ ] **Нарушение авторских прав** → загрузите только свои работы
- [ ] **Несоответствие тематике** → работа должна быть художественной
- [ ] **Некорректные метаданные** → исправьте описание и название

*После исправления вы можете повторно отправить работу на модерацию.*

### Можно ли удалить публикацию и каковы последствия?

*Да, публикацию можно удалить, но учтите следующие последствия:*

#### Непосредственные эффекты
- [x] **Работа исчезнет** из всех выставок
- [x] **Удалятся все метаданные** из поиска
- [x] **Освободится место** в вашем хранилище

#### Необратимость
- [x] **Восстановление невозможно** — данные удаляются полностью
- [x] **Рекомендуется** предварительно скачать оригинал
- [x] **Уведомляются** авторы выставок, где работа участвовала

*Перед удалением система запросит подтверждение действия.*
