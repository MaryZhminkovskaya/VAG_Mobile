## Создание выставки

Чтобы создать выставку, вам необходимо в навигаторе снизу нажать на иконку плюса и выбрать **"Создать выставку"**. После этого откроется форма создания выставки:

![Форма создания выставки](../images/create_exhibition.png)

Заполните необходимые данные:
- Название выставки
- Описание
- Тип выставки (по умолчанию публичная)

Нажмите "Создать". Поздравляем, ваша первая выставка создана!

## Типы выставок

В нашей галерее существует два типа выставок:

1. **Публичные выставки** - любой пользователь может добавлять свои работы
2. **Приватные выставки** - только вы можете добавлять работы

Подробнее о различиях между типами выставок можно узнать в [сравнительной таблице](../README.md#сравнение-типов-выставки).
