# Управление выставками

Здесь описано, как управлять уже созданными выставками.

## Редактирование выставки

Вы можете изменить параметры выставки в любое время:

1. Откройте выставку
2. Нажмите на иконку редактирования (карандаш)
3. Измените нужные параметры:
   - Название
   - Описание
   - Тип выставки (приватная/публичная)
4. Сохраните изменения

## Управление работами в выставке

**Как удалить работу из выставки:**

1. Найдите работу в списке выставки
2. Нажмите на иконку удаления (корзина)
3. Подтвердите удаление

**Важно:** при удалении работы из выставки она не удаляется из галереи и остается в профиле автора.

## Удаление выставки

Если выставка больше не нужна:

1. Откройте выставку
2. Нажмите "Настройки выставки"
3. Выберите "Удалить выставку"
4. Подтвердите действие

**Внимание:** это действие необратимо. Все работы останутся в профилях их авторов, но выставка будет удалена.
