# Добро пожаловать в Виртуальную Арт-Галерею

Добро пожаловать в документацию по приложению **"Виртуальная Арт-Галерея"** — вашему цифровому пространству для искусства.

Данная документация предлагает ознакомиться с таким объектом нашего сервиса, как **выставка**.

**Основные возможности и преимущества выставок:**

1. Организовывайте картины в удобном формате выставок.
2. Добавляйте неограниченное количество работ в выставку.
3. Публичные выставки позволяют любым пользователям добавлять работы в выставки других художников.
4. Выставки сразу добавляются в общий доступ.
5. Загруженные в выставку работы автоматически добавляются в профиль.
6. Можно добавлять уже существующие в базе данных публикации.
7. Легко редактируйте содержимое своих выставок, в том числе название, описание и сами картины.

## Сравнение типов выставки

| Функция            | Приватная выставка                                           | Публичная выставка                                                                                      |
| ------------------ | ------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------- |
| Создание           | Требует выбор пункта "Сделать выставку приватной"            | Создаётся по умолчанию                                                                                  |
| Добавление работ   | Только автор может добавлять работы                          | Любой авторизованный пользователь может добавлять работы                                                |
| Доступность        | Доступно всем                                                | Доступно всем                                                                                           |
| Редактирование     | Автору доступно редактирование выставки и своих работ        | Автору доступно редактирование выставки, своих работ и удаление чужих работ из выставки                 |
| Переключение типов | Выставка становится публичной, все данные выставки сохранены | Выставка становится приватной, все данные выставки сохранены, работы других авторов остаются в выставке |
